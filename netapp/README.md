# NetApp Integration Assets

This folder contains NetApp Trident installer assets used by Cisco AI Pods storage workflows.

## Contents

- `trident-installer/`: extracted NetApp Trident installer bundle
- `trident-installer-26.02.0.tar.gz`: packaged installer archive

## Usage Notes

1. Follow storage and OpenShift prerequisites in [../everpure/README.md](../everpure/README.md) and [../openshift/README.md](../openshift/README.md).
2. Use the Trident installer assets from this folder as required by your storage platform and target OpenShift version.
3. Keep installer versions aligned with your validated matrix before deployment.
